# Sales Intelligence System

## Overview
An end-to-end Business Intelligence project built using Excel, SQL, and Power BI to analyze sales performance and support business decision-making.

## Business Problem
The company lacked a centralized reporting solution to monitor sales, profitability, customer segments, and regional performance.

## Objectives
- Monitor Total Sales
- Monitor Total Profit
- Analyze Profit Margin
- Identify Top Products
- Identify Loss-making Products
- Compare Regional Performance
- Analyze Customer Segments
- Evaluate Discount Impact

## Tools
- Excel
- SQL
- Power BI
- GitHub

## Deliverables
- Excel Dashboard
- Power BI Dashboard
- Business Requirement Document (BRD)
- SQL Business Queries
